Installation

Follow the steps below to setup your site template:

  1. Unzip the downloaded package and open the /Gama - code folder to find all the software files. You will need to move this files to folder in your desktop for example!
  2. We are going to prepare our files for uploading in our server with the following steps:
    • Create firebase - How to create Firebase/Firestore account?
    • Creating telegram bot - How to create Telegram bot with BotFather?

      • Install visual studio code Download Visual studio Code
      • Open the project using visual studio code
      • Open new terminal
      • Run in terminal following command - npm install
      • Rebrand – Search and replace all texts.
    • npm install - this will install all neccessary node modules for your project
  3. npm run build , will create a build folder
